1# OpenAI เปิดตัว GPT-4o ฉลาดและเร็วกว่าเดิม

OpenAI เปิดตัว GPT-4o ฉลาดและเร็วกว่าเดิม (ต่อ)

กลับไปหน้าแรก
หน้าถัดไป

OpenAI ไม่หยุดยั้งในการพัฒนาเทคโนโลยี Generative AI ล่าสุดได้เปิดตัว GPT-4o โมเดลใหม่
สุดล้ำที่มาพร้อมความสามารถรอบด้านสมชื่อ Omni ไม่ว่าจะเป็นการประมวลผลข้อความ, เสียง,
ภาพ,วิดีโอ หรือแม้แต่สัญญาณต่างๆ และที่สำคัญคือทุกคนสามารถสัมผัสประสบการณ์ใหม่นี้ได้ฟรีผ่าน ChatGPT

ไม่เพียงแต่ประมวลผลข้อมูลได้เร็วขึ้นกว่า GPT-4 แต่ยังยกระดับ Voice Mode ให้
สามารถโต้ตอบได้อย่างเป็นธรรมชาติ สามารถถามคำถามขณะ ChatGPT กำลังพูด รวมถึง
แยกแยะน้ำเสียงของผู้พูดได้

Voice Mode ที่ขับเคลื่อนด้วย GPT-4o มาพร้อมฟีเจอร์ Real-Time Translation
ที่ทำหน้าที่เป็นล่ามแปลภาษาได้แบบเรียลไทม์ สามารถสร้างน้ำเสียงและอารมณ์ในการพูดได้หลากหลาย
เปลี่ยนน้ำเสียงตามคำสั่ง หรือแม้กระทั่งร้องเพลงได้ เสมือนมีผู้ช่วยอัจฉริยะส่วนตัว

อ่านรูปภาพและหน้าจอแบบเรียลไทม์ ตอบทุกข้อสงสัยได้ทันที

GPT-4o สามารถอ่านรูปภาพผ่านกล้องมือถือหรือหน้าจอคอมพิวเตอร์ได้แบบเรียลไทม์
ไม่ว่าจะติดปัญหาในการเขียนโค้ด ทำการบ้านคณิตศาสตร์ หรือแม้แต่ต้องการทราบข้อมูลของเสื้อผ้า
เพียงเปิดกล้องให้ ChatGPT ก็พร้อมตอบทุกข้อสงสัย

GPT-4o รองรับ 50 ภาษา และปรับปรุงระบบ Tokenization ให้มีประสิทธิภาพมากขึ้น

GPT-4o ได้รับการปรับปรุงให้ใช้งานได้ดีขึ้นใน 50 ภาษา และมีการปรับระบบ Tokenization เพื่อลดจำนวน
Token ที่ใช้ในการประมวลผล ทำให้ประหยัด Token ได้มากขึ้นถึง 1.1 – 4.4 เท่า ขึ้นอยู่กับภาษาที่ใช้

ChatGPT อัปเดตโมเดล GPT-4o พร้อมใช้งานแล้ววันนี้
ChatGPT ได้อัปเดตโมเดล GPT-4o ให้ใช้งานได้แล้วทั้งในรูปแบบฟรี และสำหรับผู้ใช้งาน Plus และ Team
โดยผู้ใช้งานแบบเสียเงินจะได้รับสิทธิประโยชน์เพิ่มเติม เช่น จำนวนการใช้งาน Message Limit ที่สูงกว่า
อย่างไรก็ตาม ในขณะนี้จะยังใช้งานได้เฉพาะในโหมดตัวอักษรและรูปภาพ ส่วน Voice Mode จะเปิดให้ทดสอบแบบ
Alpha สำหรับผู้ใช้งาน ChatGPT Plus ในเร็วๆ นี้
